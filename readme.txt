Create a public roadmap, collect user feedback, manage feature requests, and run an interactive ideas board in WordPress.

== Description ==

SimpleBoards helps you collect ideas and publish a clear roadmap on your own WordPress site.

People can post ideas, vote, and comment in one place, so you can see what users want most and decide faster.

When you move approved ideas to your roadmap, visitors can track progress without emailing your team for updates.

You stay in WordPress for everything, so your team has one workflow instead of switching between many tools.

More info, demos, and Pro features: https://simpleboardswp.com
Support: contact@simpleboardswp.com

= What You Can Do =

* Run a public ideas board so users can tell you what to build next.
* Let users vote and discuss ideas so demand is visible before you spend time building.
* Move approved ideas into roadmap columns so people can see what is planned, in progress, and done.
* Use filters and sorting so visitors quickly find the right items.
* Moderate posts from the WordPress admin area so low-quality items do not distract your roadmap.
* Import existing ideas from LoopedIn CSV files so you do not lose earlier feedback.

== Installation ==

1. Upload the `simpleboards-roadmap` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Go to `SimpleBoards > All Boards` and create a board
4. Add `[sbir_board product="your-board-slug"]` to any page
5. Publish the page and start collecting ideas

== Frequently Asked Questions ==

= How do I show my board on a page? =

Use `[sbir_board product="board-slug"]` and replace `board-slug` with your board slug.

= Can one board include both roadmap and ideas? =

Yes. You can show roadmap, ideas, or both on the same board.

= Can people submit ideas without login? =

Yes. Enable guest submissions in settings to collect more feedback.

= Can users filter and sort items? =

Yes. You can show filter and sort controls for both ideas and roadmap views.

= Can I import from LoopedIn? =

Yes. You can import LoopedIn CSV files and keep vote counts during migration.

= Can I control duplicates during import? =

Yes. You can allow duplicate titles or overwrite existing items with the same title.

= Is there an announcements feature? =

Yes, in Pro. You can publish board announcements in a tab and widget so users see product updates quickly.

= Can I customize board design? =

Yes. Free includes stable core styling. Pro adds full design controls for layout, colors, typography, and vote UI.

= Does it work with different WordPress themes? =

Yes. The board includes built-in styles to reduce theme conflicts and keep your board design consistent.

= Is there a Pro version? =

Yes. Pro adds private boards, announcements, workflow automations, design controls, checklist progress, and Torin AI. Details: https://simpleboardswp.com

= How do I contact the team? =

Email contact@simpleboardswp.com or visit https://simpleboardswp.com for support, demos, and Pro info.
